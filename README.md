## Create a Docker File
To create a Docker file:
- Open PowerShell in Administration mode
- Move the path that contain our project
- Run ```docker build -t dockersample -f WebApplication1/Dockerfile .```
- To check the list of available images, use this command ```docker image ls```
- To run a Docker image and create a container, use ```docker run -p 9000:80 dockersample```. Port 9000 is the external port used to run the application, while port 80 is the port inside the Container expose out according to the configuration of Dockerfile. The external 9000 port will connect to the inner port to access the application.
- To open the web application on browser, use this path: ```localhost:9000```
